# BuilderID — Endless Skill Passport PoC

A lightweight, front-end-only prototype that allows developers to:
- Connect wallet + Social DID (mocked)
- Claim skill by uploading proof (GitHub/Figma)
- Receive AI-generated summary
- Preview and “mint” skill badges as NFTs

Built with pure HTML/CSS/JS — no backend required. Deployable in seconds.

## Value to Endless
- Activates developer community with zero friction
- Creates verifiable reputation layer
- Enables future grant allocation based on proven skills
- Aligns with “Everyday Web3” — simple, visual, intuitive
